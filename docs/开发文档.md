# 端口后台代理管理系统 - 开发文档

## 系统概述

本系统是一个功能完善的多级分销卡密管理系统，基于端口代理平台二次开发，实现了代理层级管理、权限精细化控制、卡密生成与定价、充值体系等核心功能模块。系统采用PHP + MySQL架构，支持多级代理分销体系，具备完整的后台管理和代理端操作界面。

---

## 核心功能模块

### 1. 代理等级管理

**功能描述**
- 后台自定义多个代理等级
- 为每个等级设置独立的管理权限开关
- 配置各级别的卡密生成折扣率
- 设置初始余额和每日免费卡密数量
- 控制下级代理添加数量上限

**核心文件**
- `sub_admin/agent_level.php` - 等级列表管理页面
- `sub_admin/agent_level_edit.php` - 等级编辑页面
- `sub_admin/agent_level_permission.php` - 权限精细配置页面
- `install/upgrade_system.sql` - 数据库表结构

**数据库表**
```sql
CREATE TABLE `agent_level` (
  `id` int(11) NOT NULL AUTO_INCREMENT,
  `level_name` varchar(50) NOT NULL COMMENT '等级名称',
  `level_order` int(11) NOT NULL DEFAULT '0' COMMENT '等级顺序（数字越大等级越高）',
  `kami_discount` decimal(10,2) NOT NULL DEFAULT '1.00' COMMENT '卡密折扣（1.00=原价）',
  `initial_balance` decimal(10,2) NOT NULL DEFAULT '0.00' COMMENT '初始余额',
  `daily_free_kami` int(11) NOT NULL DEFAULT '0' COMMENT '每日免费卡密数量',
  `max_sub_agents` int(11) NOT NULL DEFAULT '0' COMMENT '最大下级代理数量（0=无限制）',
  `can_generate_kami` tinyint(1) NOT NULL DEFAULT '1' COMMENT '是否可以生成卡密',
  `can_manage_user` tinyint(1) NOT NULL DEFAULT '0' COMMENT '是否可以管理用户',
  `can_add_agent` tinyint(1) NOT NULL DEFAULT '0' COMMENT '是否可以添加下级代理',
  `can_set_discount` tinyint(1) NOT NULL DEFAULT '0' COMMENT '是否可设置下级折扣',
  `can_set_free_kami` tinyint(1) NOT NULL DEFAULT '0' COMMENT '是否可设置下级免费卡密数',
  `state` tinyint(1) NOT NULL DEFAULT '1' COMMENT '状态（1=启用，0=禁用）',
  `create_time` datetime NOT NULL DEFAULT CURRENT_TIMESTAMP,
  `update_time` datetime NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  PRIMARY KEY (`id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='代理等级表';
```

**权限配置项说明**
- **生成卡密权限**：控制代理是否可以生成卡密
- **用户管理权限**：控制代理是否可以管理用户账号
- **添加下级代理权限**：控制代理是否可以添加下级代理
- **设置下级折扣权限**：控制代理是否可以自定义下级代理的卡密折扣
- **设置免费卡密数权限**：控制代理是否可以自定义下级代理的每日免费卡密数量

### 2. 卡密价格管理

**功能描述**
- 设置天卡、周卡、月卡、年卡、永久卡五种类型价格
- 支持批量修改价格
- 价格启用/禁用控制
- 代理生成卡密时根据等级折扣自动计算实际扣费

**核心文件**
- `sub_admin/kami_price.php` - 卡密价格列表页面
- `sub_admin/kami_price_edit.php` - 卡密价格编辑页面

**数据库表**
```sql
CREATE TABLE `kami_price` (
  `id` int(11) NOT NULL AUTO_INCREMENT,
  `kami_type` varchar(20) NOT NULL COMMENT '卡密类型（day/week/month/year/forever）',
  `type_name` varchar(20) NOT NULL COMMENT '类型名称',
  `times` varchar(50) NOT NULL COMMENT '有效期时长',
  `price` decimal(10,2) NOT NULL COMMENT '价格',
  `sort_order` int(11) NOT NULL DEFAULT '0' COMMENT '排序',
  `state` tinyint(1) NOT NULL DEFAULT '1' COMMENT '状态（1=启用，0=禁用）',
  `create_time` datetime NOT NULL DEFAULT CURRENT_TIMESTAMP,
  `update_time` datetime NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  PRIMARY KEY (`id`),
  UNIQUE KEY `idx_kami_type` (`kami_type`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='卡密价格表';
```

**卡密类型说明**
| 类型 | kami_type | times | 说明 |
|------|-----------|-------|------|
| 天卡 | day | +1 day | 有效期1天 |
| 周卡 | week | +1 week | 有效期1周 |
| 月卡 | month | +1 month | 有效期1个月 |
| 年卡 | year | +1 year | 有效期1年 |
| 永久卡 | forever | +100 year | 永久有效 |

**价格计算公式**
```
实际扣费 = 卡密基础价格 × 代理等级折扣
```

### 6. 代理注册系统

**功能描述**
- 代理登录页面新增注册入口
- 支持两种注册方式：
  - **卡密注册**：使用卡密直接注册，卡密等级决定代理等级
  - **账号密码注册**：提交注册申请，等待后台审核
- 注册信息包含：账号、密码、姓名、联系方式、备注等

**核心文件**
- `agent/register.php` - 代理注册页面
- `sub_admin/agent_register_audit.php` - 后台注册审核页面
- `agent/ajax.php` - 注册提交接口

**数据库表**
```sql
CREATE TABLE `agent_register` (
  `id` int(11) NOT NULL AUTO_INCREMENT,
  `username` varchar(50) NOT NULL COMMENT '账号',
  `password` varchar(255) NOT NULL COMMENT '密码',
  `name` varchar(50) NOT NULL COMMENT '姓名',
  `contact` varchar(100) NOT NULL COMMENT '联系方式',
  `register_type` tinyint(1) NOT NULL COMMENT '注册方式（1=卡密，2=账号密码）',
  `kami_code` varchar(50) DEFAULT NULL COMMENT '卡密码（卡密注册时）',
  `level_id` int(11) NOT NULL COMMENT '代理等级ID',
  `remark` varchar(500) DEFAULT NULL COMMENT '备注',
  `register_time` datetime NOT NULL COMMENT '注册时间',
  `register_ip` varchar(50) NOT NULL COMMENT '注册IP',
  `audit_status` tinyint(1) NOT NULL DEFAULT '0' COMMENT '审核状态（0=待审核，1=通过，2=拒绝）',
  `audit_time` datetime DEFAULT NULL COMMENT '审核时间',
  `audit_admin_id` int(11) DEFAULT NULL COMMENT '审核管理员ID',
  `audit_remark` varchar(500) DEFAULT NULL COMMENT '审核备注',
  PRIMARY KEY (`id`),
  UNIQUE KEY `idx_username` (`username`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='代理注册申请表';
```

**卡密注册流程**
1. 输入卡密码
2. 验证卡密是否存在且未使用
3. 卡密指定等级或用户选择等级
4. 提交注册信息
5. 卡密标记为已使用
6. 注册成功

**账号密码注册流程**
1. 填写完整注册信息
2. 选择代理等级（可选）
3. 提交注册申请
4. 管理员后台审核
5. 审核通过后创建代理账号

### 7. 下级代理管理

**功能描述**
- 代理添加下级代理
- 设置下级代理等级
- 根据权限配置下级折扣和免费卡密数

**下级代理配置项**

- 账号、密码、姓名、联系方式
- 卡密折扣（如果具备权限）
- 每日免费卡密数（如果具备权限）

### 8. 代理生成卡密系统

**功能描述**
- 根据等级折扣计算实际扣费
- 每日免费卡密数量限制
- 卡密类型选择（天/周/月/年/永久）
- 卡密数量设置
- 卡密备注信息

**数据库表**
```sql
CREATE TABLE `agent_daily_kami` (
  `id` int(11) NOT NULL AUTO_INCREMENT,
  `agent_id` int(11) NOT NULL COMMENT '代理ID',
  `record_date` date NOT NULL COMMENT '记录日期',
  `free_kami_count` int(11) NOT NULL DEFAULT '0' COMMENT '免费卡密数量',
  `total_kami_count` int(11) NOT NULL DEFAULT '0' COMMENT '总生成数量',
  `total_cost` decimal(10,2) NOT NULL DEFAULT '0.00' COMMENT '总消耗金额',
  `create_time` datetime NOT NULL DEFAULT CURRENT_TIMESTAMP,
  `update_time` datetime NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  PRIMARY KEY (`id`),
  UNIQUE KEY `idx_agent_date` (`agent_id`, `record_date`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='代理每日卡密记录表';
```

**卡密生成流程**
1. 选择卡密类型和数量
2. 检查余额是否充足
3. 检查今日免费卡密剩余数量
4. 计算实际扣费金额
5. 扣除余额
6. 生成卡密并返回

**免费卡密计算**
```
如果 今日免费卡密已用 < 等级每日免费卡密数：
    实际扣费 = 0
else:
    实际扣费 = 卡密基础价格 × 等级折扣 × 数量
```

## 安装与部署

### 环境要求
- PHP 7.3 或更高版本
- MySQL 5.7 或更高版本
- Apache/Nginx Web服务器
- 支持PDO扩展

### 安装步骤

#### 1. 上传文件
将解压后的 `端口后台有代理` 文件夹上传到网站根目录

#### 2. 导入数据库
```bash
# 导入原有数据库结构
mysql -u用户名 -p数据库名 < install/ccpy.sql

# 导入升级脚本
mysql -u用户名 -p数据库名 < install/upgrade_system.sql
```

#### 3. 配置数据库连接
编辑 `config.php` 文件，配置数据库连接信息：
```php
$dbconfig = [
    'host' => 'localhost',        // 数据库地址
    'port' => 3306,                // 数据库端口
    'user' => 'root',              // 数据库用户名
    'pwd' => 'password',          // 数据库密码
    'dbname' => 'your_db_name'     // 数据库名
];
```

#### 4. 设置目录权限
```bash
chmod 755 端口后台有代理/
chmod 777 端口后台有代理/config.php
```

#### 5. 访问系统
- 后台管理地址：`http://你的域名/端口后台有代理/sub_admin/`
- 代理端地址：`http://你的域名/端口后台有代理/agent/`

#### 6. 默认账号
- 超级管理员：`admin` / `admin888`
- 请登录后立即修改密码

## 使用指南

### 后台管理操作流程

#### 1. 配置代理等级
1. 进入 `代理等级管理` 页面
2. 点击 `添加等级` 创建新等级
3. 设置等级名称、顺序、折扣、初始余额等
4. 配置该等级的各项权限开关
5. 保存

#### 2. 设置卡密价格
1. 进入 `卡密价格管理` 页面
2. 点击 `编辑` 修改各类型卡密价格
3. 点击 `启用/禁用` 控制价格是否可用
4. 保存

#### 3. 添加代理
1. 进入 `代理列表` 页面
2. 点击 `添加代理`
3. 选择代理等级
4. 填写代理信息（账号、密码、姓名等）
5. 系统自动生成专属ID
6. 保存

#### 4. 充值代理余额
1. 进入 `余额充值` 页面
2. 输入代理专属ID
3. 系统自动查询并显示代理信息
4. 输入充值金额
5. 点击 `充值`（无需支付）
6. 充值成功

#### 5. 审核注册申请
1. 进入 `代理注册审核` 页面
2. 查看待审核申请列表
3. 点击 `查看详情` 查看申请信息
4. 选择 `通过` 或 `拒绝`
5. 填写审核备注
6. 提交审核结果

#### 6. 发布系统公告
1. 进入 `代理系统公告` 页面
2. 点击 `添加公告`
3. 填写公告标题和内容
4. 选择目标等级（可选）
5. 设置置顶和过期时间（可选）
6. 保存

### 代理端操作流程

#### 1. 代理注册
**卡密注册**
1. 访问代理注册页面
2. 选择 `卡密注册` 方式
3. 输入卡密码
4. 填写账号、密码、姓名等信息
5. 提交注册

**账号密码注册**
1. 访问代理注册页面
2. 选择 `账号密码注册` 方式
3. 填写完整注册信息
4. 选择代理等级（可选）
5. 提交注册申请
6. 等待后台审核

#### 2. 登录代理端
1. 输入账号密码
2. 登录成功后进入代理管理后台

#### 3. 生成卡密
1. 进入 `卡密管理` 页面
2. 选择卡密类型（天/周/月/年/永久）
3. 输入生成数量
4. 系统自动显示预计扣费金额
5. 点击 `生成卡密`
6. 查看生成的卡密列表

#### 4. 添加下级代理
1. 进入 `下级代理管理` 页面
2. 点击 `添加下级代理`
3. 填写下级代理信息
4. 选择下级代理等级
5. 根据权限设置折扣和免费卡密数
6. 提交

#### 5. 生成下级代理卡密
1. 进入 `下级代理卡密` 页面
2. 选择下级代理等级
3. 输入生成数量
4. 系统自动计算扣费金额
5. 点击 `生成`
6. 获取卡密列表

## API接口说明

### 代理端接口

#### 1. 代理注册
**接口地址**：`agent/ajax.php?act=agentregister`

**请求方式**：POST

**请求参数**：
```json
{
  "username": "代理账号",
  "password": "密码",
  "name": "姓名",
```

1. **数据库安全**
   - 修改默认数据库账号密码

   - 配置 IP 白名单
   - 启用登录验证码

## 常见问题

### Q1: 代理注册后无法登录？
**A**: 请检查：
- 注册申请是否已审核通过
- 账号密码是否正确
- 代理账号是否被封禁

### Q2: 生成卡密时提示余额不足？
**A**: 请检查：
- 代理余额是否充足
- 卡密类型价格设置是否正确
- 代理等级折扣是否生效

### Q3: 下级代理添加失败？
**A**: 请检查：
- 是否具备添加下级代理权限
- 下级代理数量是否已达上限
- 下级代理等级是否低于当前等级

### Q4: 系统公告代理端看不到？
**A**: 请检查：
- 公告状态是否为启用
- 公告目标等级是否包含该代理等级
- 公告是否已过期

### Q5: 数据库升级失败？
**A**: 请检查：
- MySQL版本是否符合要求
- 数据库字符集是否为 utf8mb4
- 是否有足够的数据库权限
