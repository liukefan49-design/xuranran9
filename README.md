# BC Web Application

这是从 bc.tar 解压的 Web 应用项目。

## 项目结构

- `agent/` - 代理相关文件
- `api/` - API 接口文件
- `assets/` - 静态资源文件
- `cache/` - 缓存目录
- `docs/` - 文档文件
- `includes/` - 包含文件
- `install/` - 安装文件
- `logs/` - 日志文件
- `sub_admin/` - 子管理员文件

## 安装说明

请参考 `install/` 目录下的安装指南。

## 文件统计

- 总文件数: 260 个
- 主要目录: 8 个
